O que você vai aprender?

Você irá aprender a criar interfaces digitais claras, consistentes e funcionais, entendendo como organizar telas, componentes e fluxos visuais para produtos web e mobile.

Fundamentos de UI

  • Princípios de hierarquia visual, alinhamento e contraste.
  • Uso de cores, tipografia e espaçamento em interfaces.
  • Leitura de referências e criação de padrões visuais.
  • Atividade: redesenhar uma tela simples com mais clareza.

Componentes e Layouts

  • Construção de botões, cards, menus e campos de formulário.
  • Organização de grids para telas responsivas.
  • Consistência visual entre estados, variações e seções.
  • Atividade: montar uma tela de produto com componentes reutilizáveis.

Protótipo Navegável

  • Criação de fluxos entre telas principais.
  • Microinterações, estados de clique e feedback visual.
  • Preparação de protótipo para apresentação.
  • Atividade: publicar um fluxo navegável para validação.

Para quem é?

Ideal para estudantes e iniciantes da área de comunicação, design e tecnologia que desejam criar telas digitais mais profissionais, organizadas e prontas para prototipação.

Ao final do curso, o aluno terá uma base sólida para criar interfaces digitais, organizar componentes, construir telas consistentes e apresentar protótipos com mais segurança em projetos criativos e de tecnologia.
